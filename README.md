Learning MySQL as a Data Engineer
A structured, beginner-to-advanced roadmap and notes repository for learning MySQL from a data engineering perspective.
📌 About
This repository documents my journey learning MySQL, covering core SQL concepts, database design, performance optimization, and real-world data engineering use cases such as ETL pipelines, data warehousing basics, and query optimization.
🎯 Goals
Build a strong foundation in SQL and relational database concepts
Understand database design, normalization, and indexing
Write efficient queries for large datasets
Apply MySQL in real-world data engineering workflows (ETL, pipelines, automation)
Practice with hands-on projects and datasets
🗂️ Topics Covered
1. Basics
Installing and setting up MySQL
Databases, tables, schemas
Data types
CRUD operations (SELECT, INSERT, UPDATE, DELETE)
Filtering, sorting, and limiting results
2. Intermediate
Joins (INNER, LEFT, RIGHT, FULL, SELF)
Aggregations and GROUP BY / HAVING
Subqueries and CTEs
Views
Constraints and keys (Primary, Foreign, Unique)
3 . Advanced
Indexes and query optimization
Stored procedures, functions, and triggers
Transactions and ACID properties
Window functions
Query execution plans (EXPLAIN)
